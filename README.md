Proyecto WinForms con SQL Server para gestionar tareas de manera sencilla.

## Características

- Agregar tareas nuevas
- Marcar tareas como completadas con confirmación
- Borrar tareas con confirmación
- Checkboxes para indicar el estado de cada tarea
- Datos persistentes en SQL Server
- Interfaz intuitiva con DataGridView

## Tecnologías

- C# con Windows Forms
- SQL Server (LocalDB o SQLEXPRESS)
- Git y GitHub para control de versiones
